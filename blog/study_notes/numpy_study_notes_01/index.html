NumPy 学习笔记 01

NumPy 学习笔记 01

本文结合实际代码案例与教学课件，梳理 NumPy 的基础知识，包括数组的创建、属性、运算及构造方式，适合 Python 数据分析初学者。

一、NumPy 概述

  • NumPy，全称 Numerical Python，是 Python 的科学计算基础库。
  • 它补充了 Python 在数值处理方面的能力，是众多数据分析与机器学习库的基础。
  • 采用标准 C 语言实现，运算效率高，免费开源。

NumPy 的发展历史

  • 1995年：Numeric 项目开启
  • 2001年：Numarray 提供多维数组支持
  • 2005年：Numeric 与 Numarray 合并，形成 NumPy
  • 2006年：NumPy 成为独立项目

NumPy 的核心：多维数组 + 数值计算

  • 代码简洁：替代大量 for 循环操作
  • 结构高效：底层使用 C 实现数据处理，Python 提供接口

二、Numpy 基础语法

1. 创建 ndarray 数组

import numpy as np

# 从列表创建数组
arr = np.array([1, 2, 3, 4, 5])
print(arr)

数组属性

print(arr.ndim)   # 维度数
print(arr.shape)  # 形状
print(arr.size)   # 总元素数
print(arr.dtype)  # 数据类型

2. 广播机制

print(arr + 1)       # 每个元素加1
print(arr * 2)       # 每个元素乘2
print(arr == 3)      # 布尔判断

3. 多维数组操作

arr2d = np.array([[1, 2, 3], [4, 5, 6]])
print(arr2d.ndim)
print(arr2d.shape)
print(arr2d * 3)

4. 数组与矩阵的区别

arr = np.array([[1,2,3],[6,7,8],[11,12,13]])
print(arr * arr)  # 元素乘法
print(np.matrix(arr) * np.matrix(arr))  # 矩阵乘法

输出

数组乘法结果：
[[  1   4   9]
 [ 36  49  64]
 [121 144 169]]

矩阵乘法结果：
[[ 46  52  58]
 [136 157 178]
 [226 262 298]]

三、数组构造方法

1. 等差数组

np.arange(1, 10)  # 从1到9

2. 全零数组

np.zeros((3, 5))

3. 全一数组

np.ones((3, 5), dtype='int8')

4. 与已有数组结构一致的全一或全零数组

tmp = [[1, 2, 5], [8, 3, 1]]
np.ones_like(tmp)
np.zeros_like(tmp)

5. 线性间隔数组

np.linspace(-3.14, 3.14, 200)

四、正态分布随机数组生成

arr = np.random.normal(0, 1, 10000)  # 均值0，标准差1
print(arr)
